<script src="../base1/jquery.js"></script>

To use jQuery in your package include the above script. Other Cockpit and PatternFly code requires jQuery. You should always load it, if you use any of the other Cockpit javascript code. It should be the first javascript module loaded.

The current jQuery version is 2.x. In the latest package we may track relatively recent versions of jQuery. Other packages may provide more stable versions of jQuery in the future.

Also included in this file are the jQuery plugins used. For this package these these are at least: Bootstrap 3.0.3.
