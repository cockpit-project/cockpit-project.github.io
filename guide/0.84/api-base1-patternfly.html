patternfly.css — PatternFly standard style sheets

Description

<link href="../base1/patternfly.css" type="text/css" rel="stylesheet">
@import url('../base1/patternfly.css');

The standard PatternFly stylesheets. If you include cockpit.css then don't include this directly. It is firmly recommended to use PatternFly as the feel for your Cockpit component.

The version of PatternFly is an early prototype, and will be updated soon. See the PatternFly documentation for information on the widgets and classes available.
