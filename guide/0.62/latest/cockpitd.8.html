Synopsis

cockpitd [--help] [--replace] [--http-root PATH] [--no-debug] [--no-sigint]

DESCRIPTION

The cockpitd program provides the com.redhat.Cockpit name on the session message bus. Users or administrators should never need to start this daemon as it automatically started by systemd(1) on bootup.

OPTIONS

--help

Show help options.

--replace

Replace existing daemon.

--no-debug

Do not print debug or informational messages on stdout/stderr.

--no-sigint

Do not handle SIGINT for controlled shutdown.

BUGS
