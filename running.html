Cockpit comes installed by default in Fedora Server.
Cockpit can be found in the Arch User Repository as package cockpit.
 
Connect to a Fedora Atomic host from an existing host running Cockpit with the Add Server UI.
Alternatively you can install Cockpit on the Atomic host using the command sudo atomic run cockpit/ws
Connect to a Centos Atomic host from an existing host running Cockpit with the Add Server UI.
Alternatively you can install Cockpit on the Atomic host using the command sudo atomic run cockpit/ws
 
Cockpit is included in the Red Hat Enterprise Linux 7.1+ Extras repository.
Enable the repository using the command sudo subscription-manager repos --enable rhel-7-server-extras-rpms
Then install cockpit using the command sudo yum install cockpit and start it with sudo systemctl start cockpit
Cockpit is not part of Official Ubuntu repository. But user can use following PPA on Ubuntu 15.10 or later.
Enable PPA repository using the command sudo add-apt-repository ppa:jpsutton/cockpit
Then install cockpit using the command sudo apt-get install cockpit and start it with sudo systemctl start cockpit

If you already have Cockpit on your server, point your web browser to: https://ip-address-of-machine:9090
