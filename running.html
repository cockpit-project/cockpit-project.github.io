Cockpit comes installed by default in Fedora 21 Server.
 
Cockpit can be found in the Arch User Repository as package cockpit or cockpit-git.
Cockpit comes installed by default in Fedora Atomic.
Cockpit comes installed by default in CentOS Atomic.
Cockpit is included in the Red Hat Enterprise Linux 7 Extras repository.
Enable the repository with subscription-manager repos --enable rhel-7-server-extras-rpms
Then install cockpit with the command sudo yum install cockpit and start it with sudo systemctl start cockpit.socket

If you already have Cockpit on your server, point your web browser to: https://ip-address-of-machine:9090

Minimum client browser versions

Mozilla Firefox 11+
Internet Explorer 10+
Apple iOS Safari 6.1+
Google Chrome 16+
Opera 21.1+
Android Browser 4.4+
