Cockpit

If you already have Cockpit on your server, point your web browser to: https://ip-address-of-machine:9090

 
Connect to a Fedora or CentOS Atomic host from an existing host running Cockpit with the Add Server UI.
Alternatively you can install Cockpit on the Atomic host using the command sudo atomic run cockpit/ws
 
Cockpit comes installed by default in Fedora Server.
 
Cockpit is included in the Red Hat Enterprise Linux 7.1+ Extras repository.
Enable the repository using the command sudo subscription-manager repos --enable rhel-7-server-extras-rpms
Then install cockpit using the command sudo yum install cockpit and start it with sudo systemctl start cockpit
Cockpit is not part of the official Debian repositories, but you can add a repository which contains weekly builds:
  1. Add the line deb https://fedorapeople.org/groups/cockpit/debian unstable main to /etc/apt/sources.list
  2. Import Cockpit's signing key to the apt sources keyring: sudo apt-key adv --keyserver sks-keyservers.net --recv-keys F1BAA57C
  3. Update package information with that source with sudo apt-get update
  4. Install cockpit with sudo apt-get install cockpit

Cockpit can be found in the Arch User Repository as package cockpit.
Cockpit is not part of the official Ubuntu repositories. You can install it from the Cockpit PPA by executing the following commands:
  1. sudo add-apt-repository ppa:cockpit-project/cockpit
  2. sudo apt-get update
  3. sudo apt-get install cockpit
